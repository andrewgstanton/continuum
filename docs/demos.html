Continuum Demos

These demos show Continuum’s core ideas in action: local authorship, signed proof, portable identity, and publishing as a separate step.

Some demos are simple videos. Others are interactive flows you can try directly.

Manual Nostr Auth Demo

This demo shows how a website can authenticate a user with a Nostr public key without creating a password, storing a username, or receiving a private key.

The page requests a challenge payload. A compatible signer signs the challenge locally. The API verifies the proof and creates a session.
