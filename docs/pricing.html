Continuum Pricing

Continuum is software you download and run on your own machine. There is no required subscription and no hosted account.

How Continuum is offered

Continuum is not a hosted platform. It is software you run yourself.

You can download it from the downloads page and run it locally.

Continuum is currently distributed under license during early releases. Patron contributions help fund continued development while the project moves toward a future open-source release.

The pricing below is for optional walkthroughs, advisory sessions, and patron contributions.
