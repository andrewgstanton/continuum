Continuum Overview

Continuum is a local-first publishing environment designed to support durable authorship and self-custodied identity.

Local First

Continuum runs locally on your own machine. Your identities, keys, and publishing environment remain under your control.

Identity Management

Continuum allows you to manage multiple identities (npubs) and publish content using whichever identity you choose.

Publishing Model

Instead of publishing directly through a centralized service, Continuum separates the publishing process into steps:

  • Create content locally
  • Sign events locally
  • Publish intentionally

Durable Archive

All published content is stored locally as part of your archive. This creates a durable record of your work that is independent of any specific platform.

Getting Started

To begin using Continuum:

  1. Download the package from the downloads page
  2. Install Docker if needed
  3. Run the included startup script
  4. Open the dashboard locally
