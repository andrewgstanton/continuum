Continuum Tools

These are two browser-based verification tools included with Continuum. They run entirely in your browser, require no server requests, and work fully offline — you do not need to have Continuum installed to use them.

You can use these tools to verify Nostr events and signed artifact bundles (ZIP files with artifacts, manifest, signers, and proofs) directly in your browser.

To use Verify Signed Artifact Bundle, the ZIP must have been created with the Sign Existing Artifact feature inside a local Continuum installation. Each signed artifact bundle is cryptographically tied to your private keys, and the ZIP is generated and packaged entirely within your own Continuum workspace.
