Manual Nostr Auth Demo

This page demonstrates passwordless authentication using a Nostr public key.

The website requests a challenge payload. You paste that payload into a compatible signing tool. The tool extracts the challenge string, signs it, and returns a verification payload.

No password is created. No username is required. No private key is sent to the server.

1. Enter your Nostr public key

Enter an npub or 64-character hex public key.

2. Copy the challenge payload

Copy this full payload and paste it into a compatible signing tool.

The signing tool should extract the challenge string internally and sign that exact string.

3. Paste signed verification payload

Paste the signed verification payload returned by your signer.

Expected format: 

{
  "pubkey": "...",
  "challenge": "...",
  "signature": "..."
}

4. Session check

After a successful verification, check whether the API recognizes the browser session.
