Download Continuum

Continuum is a local-first publishing and identity environment built around self-custody, durable authorship, and sovereignty-first workflows.

Choose Your Platform

Mac App

Native macOS application bundle.

Open Mac Release Folder (v1.6.1)

Mac install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Optional: download the verify checksum file
  4. Move Continuum.app to Applications or Downloads
  5. Double-click to launch
Mac first-time launch help
  1. Right-click Continuum.app
  2. Select Open
  3. Click Open again when prompted

Windows App

Native Windows executable.

Open Windows Release Folder (v1.6.1)

Windows install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Optional: download the verify checksum file
  4. Double-click the executable
  5. Allow SmartScreen if prompted

Docker Package

For advanced users and server environments.

Open Docker Release Folder (v1.6.0)

Docker install instructions
  1. Download and unzip the package
  2. Install Docker Desktop
  3. Optional: downlad the verify checksum file
  4. Run ./run.sh
  5. Open the dashboard locally

Verify Download (Optional)

Checksum verification

Mac:

shasum -a 256 filename.zip

Windows (PowerShell):

Get-FileHash filename.zip -Algorithm SHA256

Compare the output to the checksum file in the release folder.
