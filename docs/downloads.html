Download Continuum

Continuum is designed around sovereignty-first workflows: your identities, keys, drafts, and archives remain under your control and run locally on your own machine.

Native application bundles may be signed and notarized to reduce operating system warnings and improve installation trust. This packaging step is for user safety and ease of installation — it does NOT change the local-first custody model or require any hosted account.

The codebase is currently distributed under license during early releases. The long-term intention remains to fully open source the project.

Choose Your Platform

Mac App

Native macOS application bundle (dmg).

Open Mac Release Folder (v1.6.7.4)

Mac install instructions
  1. Download the zip or dmg file
  2. Unzip the archive (if applicable)
  3. double click the dmg (if applicable)
  4. Optional: download the verify checksum file
  5. Move Continuum.app to Applications or Downloads
  6. Double-click to launch
Mac first-time launch help
  1. Right-click Continuum.app
  2. Select Open
  3. Click Open again when prompted
  4. NOTE: Versions 1.6.6 and beyond are signed dmgs you only need to accept opening an app/dmg from the internet

Windows App

Native Windows executable.

Open Windows Release Folder (v1.6.7.4)

Windows install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Optional: download the verify checksum file
  4. Double-click the executable
  5. Allow SmartScreen if prompted
Windows first-time launch help
  1. Click More info
  2. Click Run anyway

New! Local (Linux/Unix/Posix) compatible build - Docker NOT required

Open Local Release Folder (v1.6.7.4)

Local install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Optional: download the verify checksum file
  4. In terminal session navigate to dashboard folder in the unzipped package (cd ~/Downloads/Continuum-local/dashboard)
  5. chmod +x *.sh
  6. ./run_local.sh
  7. NOTE: you will need python3 or later installed in your linux or MacOS environment.

Docker Package

For advanced users and server environments.

Open Docker Release Folder (v1.6.7.4)

Docker install instructions
  1. Download and unzip the package
  2. Install Docker Desktop
  3. Optional: downlad the verify checksum file
  4. Run ./run.sh
  5. Open the dashboard locally
