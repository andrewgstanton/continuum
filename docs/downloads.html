Continuum Downloads

Continuum is a local-first publishing and identity environment built around self-custody and durable authorship.

What you can do with it

  • Manage multiple identities (npubs)
  • Publish notes and long-form articles
  • Maintain a durable local archive
  • Work in a sovereignty-first workflow (sign locally, publish intentionally)

Install (high level)

  1. Download the ZIP package from the current release folder
  2. Unzip the archive
  3. Install Docker if it is not already installed
  4. Run ./run.sh from the dashboard folder
  5. Open the dashboard locally in your browser

Verify Integrity

It is recommended to verify the integrity of the download before running it. 

shasum -a 256 continuum-pro-test-v1.5.zip

Compare the output with the checksum file provided in the current release folder.
