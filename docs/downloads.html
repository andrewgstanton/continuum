Mac install instructions
  1. Download the dmg file
  2. Double click the dmg
  3. Drag Continuum.app into Applications
  4. Double-click to launch
Mac first-time launch help
  1. Right-click Continuum.app
  2. Select Open
  3. Click Open again when prompted

Windows (full release files)

Version 1.6.7.5

Open Windows release folder

Windows install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Double-click the executable
  4. Allow SmartScreen if prompted
Windows first-time launch help
  1. Click More info
  2. Click Run anyway
