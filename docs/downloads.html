Download Continuum

Continuum is for people who want to control their own work — not rely on platforms or accounts to keep it.

Continuum runs on desktop computers (Mac and Windows). It does not run on iPhone or Android.

⬇️ Download for Mac ⬇️ Download for Windows

Current version: 1.6.8.0

Still unsure about Continuum? Read this →

After downloading, open the file. On Mac, drag the Continuum icon into the Applications folder. On Windows, double-click the file and follow the installer.

Downloading Continuum gives you a tool to publish your own work — it does not provide access to this site or its content.

Advanced downloads (optional)

Mac (full release files)

Version 1.6.8.0

Open Mac release folder

Mac install instructions
  1. Download the dmg file
  2. Double click the dmg
  3. Drag Continuum.app into Applications
  4. Double-click to launch
Mac first-time launch help
  1. Right-click Continuum.app
  2. Select Open
  3. Click Open again when prompted

Windows (full release files)

Version 1.6.7.8

Open Windows release folder

Windows install instructions
  1. Download the zip file
  2. Unzip the archive
  3. Double-click the executable
  4. Allow SmartScreen if prompted
Windows first-time launch help
  1. Click More info
  2. Click Run anyway

Local Package (advanced)

Version 1.6.8.0

For Linux, macOS, and local development environments. Python 3 required.

Open Local release folder

Local install instructions
  1. Download and unzip the package
  2. Open a terminal in the project folder
  3. Run ./run_local.sh

Docker Package (advanced)

Version 1.6.8.0

For advanced users and server environments.

Open Docker release folder

Docker install instructions
  1. Download and unzip the package
  2. Install Docker Desktop
  3. Run ./run.sh
